# SolidTypescript

Este repositório contém exemplos práticos dos princípios **SOLID** aplicados com **TypeScript**.

Cada pasta dentro de `src/` representa um dos cinco princípios da programação orientada a objetos:

- **S** – Responsabilidade Única  
- **O** – Aberto/Fechado  
- **L** – Substituição de Liskov  
- **I** – Segregação de Interface  
- **D** – Inversão de Dependência  

O objetivo é demonstrar de forma clara e organizada como aplicar esses princípios no desenvolvimento de software com TypeScript, melhorando a estrutura, manutenção e escalabilidade do código.
